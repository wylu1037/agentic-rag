## Definition of Done（第 0 周）

满足以下条件即可判定第 0 周完成：

1. 首批场景、用户、范围与非目标已冻结。
2. KPI 定义与阈值已确认并可执行。
3. 已形成 50-100 条问答评测集（当前已提供 50 条 seed，可继续扩展）。
4. 每条评测题均能追溯到文档来源并可人工复核。

## 当前假设

当前默认场景为“Agentic RAG 项目内部知识问答助手”。如果后续改为业务领域（客服、法务、医疗等），请优先更新 `scope.md` 与 `kpi.md`。
